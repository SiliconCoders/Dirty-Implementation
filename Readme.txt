	MavChat
**************************
Quick and Dirty Implementation -

->Here we have created a Real time chat room web application.
->We used PHP and WampServer.

Functionalities -

->As it is a Quick and Dirty implementation we have just created a Real time chat room application.
->User can submit his name, enter into the chat room and he can send and view the messages.
->He can exit the chat room.

Steps to Execute -

->Download and Install Wampserver from "http://www.wampserver.com/en/"
->Copy all the files From zip folder and paste it in wampserver/www/mavchat
->Now open web browser and type "localhost/mavchat/login.html"
->Now you can type your name and enter into the chat room.
